# 🩺 AI-Based Medical Diagnosis and Disease Prediction System

## 📌 Project Description
This project is an AI-powered medical diagnosis system that predicts possible diseases based on user-input symptoms using Machine Learning. It provides real-time predictions along with confidence scores through an interactive Streamlit web application.

---

## 🚀 Features
- Predicts diseases based on symptoms (Fever, Cough, Headache, Fatigue)
- Uses Decision Tree Machine Learning algorithm
- Displays prediction confidence score
- Provides basic health suggestions
- User-friendly interface built with Streamlit

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

---

## 🧠 Machine Learning Model
- Algorithm: Decision Tree Classifier
- Type: Supervised Learning
- Input: Symptoms (Yes/No)
- Output: Predicted Disease

---

## ▶️ How to Run the Project

### 1. Install dependencies
pip install streamlit scikit-learn pandas

### 2. Run the application
python -m streamlit run medical_ai.py

---

## 📊 Example Input
- Fever: Yes  
- Cough: Yes  
- Headache: No  
- Fatigue: Yes  

👉 Output: Flu (with confidence score)
